# Gestion de budget

Ce projet est un programme Python permettant de gérer un budget personnel de manière simple.

## Fonctionnalités
- Choisir un type de dépense
- Vérifier si le budget est suffisant
- Accepter ou refuser la dépense
- Calculer le montant restant après chaque opération
- Menu avec plusieurs choix

## Objectif
Ce projet a été réalisé dans le cadre de mon apprentissage en développement informatique.  
Il m’a permis de pratiquer les conditions, les boucles et la logique en Python.

## Langage utilisé
- Python
